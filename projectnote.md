# DEVELOPMENT PROCESS
- install long-time-support Node js from the Node js main page.
- npm init -y to derive a package.json file.
- npm i express to install express.
- npm -i --save-dev nodemon to install nodemon as a dev dependency to reload without restarting.
- Import express to app file.
- Initialize express.
- Define port.
- Use get to define and connect with routes.
- Use the views folder to render HTML-like pages using ejs for now
- Use set to use ejs files from views.
- Use the routes folder to handle routes.
- Use a response.render to render index file
- Create three routes, for now, admin, products, and users
- User route handles '/users', '/users/profile', '/users/cart'
- Use a router to connect with express in app.js
- Export router
- In app.js import specific router 
- Use app.use to use the imported router
- Deal with the commit issue
